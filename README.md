# CloudKraftFE

Frontend repo for CloudKraft

## Branches - 
- branches for all the team members
- `develop branch` - used for integraiton
- `main branch` - stable branch and used for deployment

## Commands 

- `npm test` - run the tests locally before committing
- `npm run build` - builds the project locally to check if Github actions pass or fail
- `npm run dev`- runs the project
